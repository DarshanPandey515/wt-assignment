# WT Assignment – Website Creation

## Overview

This repository contains the Web Technology (23BCA6C02) assignment developed as part of the Sixth Semester BCA coursework.

The project demonstrates the creation of a structured multi-page website using HTML, CSS, and JavaScript with proper navigation, layout design, and basic interactivity.

---

## Objective

The objective of this project is to:

- Understand and implement semantic HTML structure
- Design clean layouts using CSS
- Apply JavaScript for basic interactivity
- Build a multi-page website with proper navigation
- Improve practical knowledge of front-end web development

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- TailwindCss
- Visual Studio Code

---

## How to Run the Project

1. Clone the repository:
   ```
   git clone https://github.com/your-username/wt-assignment.git
   ```

2. Open the project folder.

3. Open `index.html` in any web browser.

No additional setup or installation is required.
